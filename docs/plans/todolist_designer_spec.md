# TodoList Designer 기획서

## 1. 개요

### 1-1. 프로젝트 목적
- 사용자가 할 일을 빠르게 추가하고 잊어버리기 전에 즉시 기록할 수 있는 앱
- 마감일과 우선순위 기반으로 중요한 일을 먼저 처리할 수 있도록 지원
- 완료 체크를 통해 성취감을 느끼며 진행 상황을 파악

### 1-2. 타겟 사용자
- 일반 사용자 (개인 할 일 관리)
- 업무/개인/학습 등 영역별 할 일 분류가 필요한 사용자

### 1-3. 핵심 가치
- **빠른 기록**: 할 일을 즉시 추가할 수 있는 간편한 UX
- **우선순위 관리**: 중요도에 따른 시각적 구분
- **성취감**: 완료 체크를 통한 진행 상황 시각화

## 2. 사용자 스토리

| 역할 | 목표 | 이유 | 관련 화면 |
|------|------|------|-----------|
| 일반 사용자 | 할 일을 빠르게 추가하고 싶다 | 잊어버리기 전에 즉시 기록할 수 있다 | 홈, 추가 화면 |
| 일반 사용자 | 마감일과 우선순위를 설정하고 싶다 | 중요한 일을 먼저 처리할 수 있다 | 추가/편집 화면 |
| 일반 사용자 | 완료한 할 일을 체크하고 싶다 | 성취감을 느끼며 진행 상황을 파악할 수 있다 | 홈, 완료 목록 |
| 일반 사용자 | 카테고리(태그)로 할 일을 분류하고 싶다 | 업무/개인/학습 등 영역별로 관리할 수 있다 | 홈, 추가 화면 |
| 일반 사용자 | 마감 전 알림을 받고 싶다 | 기한을 놓치지 않을 수 있다 | 마이페이지 |

## 3. 유저 플로우

### 3-1. 전체 플로우

```
[앱 실행] -> [온보딩] -> [로그인] -> [홈(할 일 목록)]
                                         |
    [마이페이지] <- [완료 목록] <- [할 일 상세] <- [할 일 추가/편집]
```

### 3-2. 주요 플로우 상세

#### 신규 사용자 플로우
```
[앱 최초 실행]
    |
[온보딩 화면]
    | 슬라이드 스와이프 (3페이지)
    | 시작하기 버튼 탭
    v
[로그인 화면]
    | 이메일/PW 입력 또는
    | Google/Apple 소셜 로그인
    v
[홈 화면] <- 로그인 성공
```

#### 할 일 추가 플로우
```
[홈 화면]
    | FAB(+) 탭
    v
[추가 화면]
    | 제목 입력 (필수)
    | 메모 입력 (선택)
    | 날짜 피커로 마감일 선택
    | 우선순위 버튼 선택 (높음/보통/낮음)
    | 태그 칩 선택
    | 저장 버튼 탭
    v
[홈 화면] <- 목록에 새 할 일 반영
```

#### 할 일 완료 플로우
```
[홈 화면]
    | 체크박스 탭
    v
[할 일 카드] <- 취소선 적용
    | 자동 이동
    v
[완료 목록] <- 완료 날짜 기준 정렬
```

#### 완료 복원 플로우
```
[완료 목록]
    | 복원 아이콘(↩) 탭
    v
[홈 화면] <- 미완료 목록으로 복원
```

## 4. 정보 구조 (IA)

```

|   |   +-- [Button] 수정 / 삭제 / 완료
|   |
|   +-- 할 일 추가/수정 (FAB 또는 수정 버튼)
|       +-- 제목 입력, 날짜 피커, 우선순위, 태그
|
+-- [Tab] 검색
|   +-- 키워드 검색
|   +-- 태그 필터
|   +-- 날짜 필터
|   +-- 미완료/완료 통합 검색
|
+-- [Tab] 완료
|   +-- 완료된 할 일 리스트 (완료 날짜 기준 정렬)
|   +-- [Action] 복원, 영구 삭제, 전체 삭제
|   +-- [Gesture] 스와이프 액션
|
+-- [Tab] 마이
    +-- 프로필
    |   +-- 닉네임, 프로필 이미지 변경
    |   +-- 소셜 연동 정보 표시
    |
    +-- 알림 설정
    |   +-- 전체 알림 ON/OFF
    |   +-- 알림 시간 설정
    |
    +-- 테마 설정 (Could Have)
    |   +-- 라이트/다크/시스템 기본값
    |
    +-- 계정 관리
        +-- 비밀번호 변경
        +-- 회원 탈퇴
        +-- 로그아웃
```
