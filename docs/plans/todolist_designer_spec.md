# TodoList Designer 기획서

## 1. 개요

### 프로젝트 목적
사용자가 할 일을 빠르게 추가하고, 마감일/우선순위를 설정하여 효율적으로 관리할 수 있는 TodoList 앱

### 타겟 사용자
- 일반 사용자 (개인 할 일 관리)
- 업무/개인/학습 등 다양한 영역의 할 일을 관리하고자 하는 사용자

### 핵심 가치
- 빠른 할 일 추가
- 직관적인 우선순위/마감일 관리
- 성취감을 주는 완료 체크 경험

---

## 2. 사용자 스토리

| 역할 | 목표 | 이유 | 관련 화면 |
|------|------|------|-----------|
| 일반 사용자 | 할 일을 빠르게 추가하고 싶다 | 잊어버리기 전에 즉시 기록할 수 있다 | 홈, 추가 |
| 일반 사용자 | 마감일과 우선순위를 설정하고 싶다 | 중요한 일을 먼저 처리할 수 있다 | 추가/수정 |
| 일반 사용자 | 완료한 할 일을 체크하고 싶다 | 성취감을 느끼며 진행 상황을 파악할 수 있다 | 홈, 완료 목록 |
| 일반 사용자 | 카테고리(태그)로 할 일을 분류하고 싶다 | 업무/개인/학습 등 영역별로 관리할 수 있다 | 추가/수정, 홈 |
| 일반 사용자 | 마감 전 알림을 받고 싶다 | 기한을 놓치지 않을 수 있다 | 마이페이지 |

## 3. 유저 플로우

### 3-1. 전체 플로우

```
[앱 실행] -> [온보딩] -> [로그인] -> [홈(할 일 목록)]
                                         |
[마이페이지] <- [완료 목록] <- [할 일 상세] <- [할 일 추가/편집]
```

### 3-2. 주요 플로우 상세

#### 할 일 추가 플로우
```
[홈 화면]
    | FAB(+) 탭
[추가 화면]
    | 제목/메모/마감일/우선순위/태그 입력
    | 저장 버튼 탭
[홈 화면] <- 목록에 즉시 반영
```

#### 할 일 완료 플로우
```
[홈 화면]
    | 체크박스 탭
[할 일 카드] <- 취소선 적용
    | 완료 목록으로 자동 이동
[완료 목록]
    | 복원(↩) 탭
[홈 화면] <- 미완료로 복원
```

#### 할 일 수정/삭제 플로우
```
[홈 화면]
    | 카드 탭
[상세 화면]
    | 수정 버튼 탭
[수정 화면] <- 기존 데이터 prefill
    | 저장 탭
[홈 화면] <- 변경사항 반영
```

## 4. 정보 구조 (IA)

```
App
├── 홈
│   ├── [Tab] 오늘 / 전체 / 태그별
│   ├── [진행률 바] 완료 현황
│   ├── [할 일 카드 리스트]
│   │   └── 체크박스, 제목, 마감일, 우선순위
│   └── [FAB] + -> 추가 화면
├── 검색
│   ├── 키워드 검색
│   ├── 태그 필터
│   └── 날짜 필터
├── 완료
│   ├── 완료된 할 일 리스트
│   ├── 복원 액션 (↩)
│   ├── 스와이프 삭제
│   └── 전체 삭제
└── 마이
    ├── 프로필 (닉네임, 이미지)
    ├── 알림 설정 (ON/OFF)
    ├── 테마 설정 (라이트/다크/시스템)
    └── 계정 관리 (비밀번호, 로그아웃, 탈퇴)
```

## 7. 인터랙션 명세

| 화면 | 인터랙션 | 피드백 |
|------|----------|--------|
| 온보딩 | 슬라이드 스와이프 | 페이지 도트 업데이트 |
| 로그인 | 로그인 버튼 탭 | 로딩 스피너 -> 성공/실패 토스트 |
| 홈 | 체크박스 탭 | 체크 애니메이션 + 취소선 + 완료 목록 이동 |
| 홈 | 카드 탭 | 리플 이펙트 -> 상세 화면 전환 |
| 홈 | FAB 탭 | 스케일 애니메이션 -> 추가 화면 |
| 추가 | 저장 탭 | 로딩 -> 성공 토스트 -> 홈 복귀 |
| 추가 | 뒤로가기 (변경사항 있을 때) | "변경사항을 저장하지 않겠습니까?" 다이얼로그 |
| 상세 | 삭제 탭 | "정말 삭제하시겠습니까?" 확인 다이얼로그 |
| 완료 | 복원 아이콘 탭 | 카드 페이드아웃 -> 홈 목록 반영 |
| 완료 | 스와이프 삭제 | 빨간 배경 노출 -> 삭제 완료 토스트 |
| 마이 | 알림 토글 | 즉시 ON/OFF 반영 |
| 마이 | 로그아웃 탭 | 확인 다이얼로그 -> 로그인 화면 |
